# @dzupagent/eval-contracts

Neutral type contracts shared between @dzupagent/evals and @dzupagent/server. Types only — no runtime dependencies.

Part of the [DzupAgent](../../README.md) framework.

## Usage

```ts
import type { BenchmarkRunRecord, EvalRunRecord, EvalSuite } from '@dzupagent/eval-contracts'
```
